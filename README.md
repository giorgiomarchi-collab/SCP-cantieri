# SCP Cantieri

App web per la gestione cantieri di **SCP s.r.l. Ingegneria e Architettura**.

Sviluppata per uso interno dello studio — Direzione Lavori e Coordinamento Sicurezza.

## Funzionalità

- Dashboard cantieri con KPI e stato avanzamento
- Schede progetto con figure professionali
- Verbali Direzione Lavori
- Verbali Sicurezza (CSE)
